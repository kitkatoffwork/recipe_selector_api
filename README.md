# Recipe Selector API

API to suggest a menu for dinner

## Getting Started

### Gitpod

1. Launch Gitpod
2. Execute the command

```
$ go run server.go
```

3. access localhost:8080

### docker-compose

1. start server

```
$ docker-compose up -d
```

2. access localhost:8080
